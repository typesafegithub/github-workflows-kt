data class CustomAction(val actionOwner: String, val actionName: String, val actionVersion: String, val inputs: Map<String, String> = emptyMap()) : RegularAction<Action.Outputs>

CustomAction can be used when there is no type-safe binding action and a quickly untyped binding is needed to fill the blank.
